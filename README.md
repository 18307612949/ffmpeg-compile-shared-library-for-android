# ffmpeg-for-android-shared-library
移植ffmpeg到android，编译可用于jni调用的so库.
本项目使用的是ffmpeg 2.6.2版本

# 
``
git clone git@github.com:dxjia/ffmpeg-for-android-shared-library.git
``

#编译
``
./build_andrioid.sh
``
